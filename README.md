# Анализ продаж интернет-магазина

Анализ датасета продаж компании (2003–2005). Определены ключевые категории товаров, прибыльные месяцы, топ-клиенты и динамика выручки.

## Используемые технологии

- **SQL** (PostgreSQL) — группировки, оконные функции, JOIN
- **Python** (pandas, matplotlib, seaborn) — очистка и визуализация

## Вопросы и выводы

1. **Самая прибыльная категория** — Classic Cars ($3.9M из $11.5M)
2. **Самый прибыльный месяц** — ноябрь (Чёрная пятница). Выручка в ноябре в 4 раза выше среднего
3. **Топ-клиент** — Euro Shopping Channel ($912K)
4. **Динамика** — пики в ноябре каждого года, падения в декабре-январе
5. **Средний чек** — выше всего у Classic Cars ($4K)
